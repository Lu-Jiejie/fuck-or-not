# Fuck or Not

基于 LLM （大语言模型）的"上不上"图片分析网站，基于你的 prompt 与图片，让 AI 分析~~色情~~图片、生成小说或故事。

一切都基于你的 prompt 。

## 使用

访问 [Fuck or Not](https://fuckornot.netlify.app/)。

1. 选择分析模式（简洁、详细、小说，或者自定义你的prompt）
2. 上传 `图片文件` 或输入图片 `外部链接`
3. 点击「分析」按钮，等待结果

你可以收藏满意的分析结果，它们会保存在浏览器的 `IndexedDB` 中，可在收藏夹页面查看。

推荐输入 `外部链接` 来使用，以节省浏览器占用与 webdav 存储占用。

## 主页/分析页

- 根据每张图配置额外提示词，帮助AI更好理解图片内的元素。（比如描述一下图片内人物的衣着~~与你想要的姿势~~）
- 人为修正返回的结果
- 保存结果到收藏夹

## 收藏夹

- 支持按时间「最新」或「最旧」排序
- 支持编辑结果

## 设置

- 自定义供应商（可使用中转站）与模型列表
- 可为自定义 Prompt ，描述图片或生成小说
- 可将所有设置（API 密钥、模型列表、Prompt 配置、收藏列表）导出为zip文件备份，或从备份文件导入设置
- 支持通过 WebDAV 协议同步设置与收藏夹数据，需要支持 CORS
