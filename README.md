# Fuck or Not

基于 LLM （大语言模型）的"上不上"图片分析网站，基于你的 prompt 与图片，让 AI 分析~~色情~~图片、生成小说或故事。

一切都基于你的 prompt 。

## 使用

访问 [Fuck or Not](https://fuckornot.netlify.app/)。

1. 选择分析模式（简洁、详细、小说，或者自定义你的prompt）
2. 上传图片
3. 点击「分析」按钮，等待结果

你可以收藏满意的分析结果，它们会保存在浏览器的 `IndexedDB` 中，可在收藏夹页面查看。

## 主页/分析页

- **额外提示词**：根据每张图配置额外提示词，帮助AI更好理解图片内的元素。（比如描述一下图片内人物的衣着~~与你想要的姿势~~）
- **编辑结果**：人为修正返回的结果
- **保存到收藏夹**：保存结果到收藏夹内

## 收藏夹

- **排序**：支持按时间「最新」或「最旧」排序
- **分页跳转**：支持直接输入页码跳转
  ~~- **导入/导出**：可将收藏导出为 JSON 文件备份，或从备份文件导入收藏~~
- **导入/导出**：导入导出功能与设置的导入导出合并

## 设置

在设置页填入你的 API 密钥与 API 地址（可选，留空则使用官方 API），支持 **Gemini**、**Grok** 以及 **ChatGPT**。

- **自定义 Prompt**：可为各个模式自定义 Prompt，或添加全新的自定义 Prompt
- **自定义模型列表**：可添加、编辑或删除模型。
- **导入/导出设置**：可将所有设置（API 密钥、模型列表、Prompt 配置、收藏列表）导出为zip文件备份，或从备份文件导入设置
- **WebDAV 同步**：支持通过 WebDAV 协议同步设置与收藏夹数据，需要支持 CORS
